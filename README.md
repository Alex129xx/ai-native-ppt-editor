# AI Native PPT Editor

一个面向 AI 生成工作流重新设计的演示文稿编辑器方案。

## 一句话概括

AI Native PPT 不应该只是“传统 PPT 编辑器 + AI 生成按钮”。它应该成为一个“叙事结构驱动的演示工作台”：用户调整受众、核心观点、章节顺序、证据权重和表达语气，AI 负责把这些意图稳定映射成幻灯片内容、布局和视觉表达。

## 推荐阅读顺序

1. [exports/final-report.md](exports/final-report.md)
   面向路演、线上展示和评审阅读，包含背景、用户洞察、产品主张、核心交互、Demo 说明和价值总结。

2. [prototype/index.html](prototype/index.html)
   打开后直接进入编辑器界面，展示“叙事大纲 Canvas + 幻灯片预览 + AI 意图控制面板”的核心体验。

3. [proposal.md](proposal.md)
   更偏产品文档，包含目标用户、痛点、MVP 范围、非目标、完整用户动线、指标和兜底机制。

## 核心亮点

- 从逐页编辑转向叙事结构编辑。
- 用“叙事大纲”驱动幻灯片内容与版式。
- 支持意图锁定，避免 AI 改坏已经确认的页面。
- 支持局部再生成，只改选中的页、章节或内容块。
- 保留来源追溯、版本回退、手动编辑和传统细节编辑兜底。

## 项目边界

这是一个面向课程展示和产品验证的静态原型，不包含真实 AI 模型调用、后端服务、账号系统、云同步或完整 PPT 编辑能力。
